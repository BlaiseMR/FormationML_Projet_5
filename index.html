I) Commandes Git

$[git init] : Initialiser le code sur Git

$[git config --global user.name 'USER'] : Identification sur GitBash

$[git config --global user.email 'E-MAIL] : Identification par e-mail

$[git add 'FILE'] : Ajoute FILL à la Liste des fichiers à partager

$[git status] : Liste des fichiers à partager

$[git rm 'FILE'] : Retire FILL de la liste

$[git add .] : Ajoute tous les fichiers du dossier à la liste

$[git rm -r --cahced .] : Retire tous les fichiers de la liste

$[git commit] : Partager la liste de fichiers

$[git commit -m 'TEXT'] : Fait un commit avec le commentaire TEXT

$[git branch NAME] : Créer une nouvelle branch NAME

$[git ckeckout BRANCH] : Change de branche

$[git branch --list] : Regarde l'ensemble des listes

$[git merge BRANCH] : Fusionne la branche actuelle avec BRANCH

$[git remote add origin https://github.com/BlaiseMR/FormationML_Projet_5.git] : Ajout de l'espace internet à Git

$[git branch -M BRANCH] : Ajoute la branche BRANCH a la liste des branch à enregistrer

$[git push -u origin BRANCH] : Enregistre dans GitHub BRANCH

$[git push] : Enregistrer les changements des commit sur Git Hub

$[git pull] : Dans le cadre d'un travail collaboratif, met à jour le GitBash avec la dernière version

$[git clone ADRESS] : Recupère un clone du fichier venant de ADRESS
